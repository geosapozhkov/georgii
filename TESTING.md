# Инструкция по тестированию

## Запуск локального сервера

Для тестирования сайта необходимо запустить локальный сервер, так как браузеры блокируют загрузку файлов через `file://` протокол из-за политики CORS.

### Вариант 1: Python
```bash
cd /Users/n/Desktop/GOSHA
python3 -m http.server 8000
```
Затем откройте в браузере: http://localhost:8000

### Вариант 2: Node.js (если установлен)
```bash
cd /Users/n/Desktop/GOSHA
npx http-server -p 8000
```

### Вариант 3: PHP (если установлен)
```bash
cd /Users/n/Desktop/GOSHA
php -S localhost:8000
```

## Тестирование анимации логотипа

1. Откройте консоль браузера (F12 или Cmd+Option+I)
2. Наведите курсор на логотип "Georgii" в левом верхнем углу
3. В консоли должны появиться сообщения:
   - "Изображения не загружены, загружаем..."
   - "Загружено изображений для лого: X"
   - "Запускаем анимацию с X изображениями"
4. На экране должны появиться изображения, которые двигаются и меняются каждые 4 секунды

## Проверка файлов

Убедитесь, что в папке `images/logo-gallery/` есть:
- Файл `files.json` со списком изображений
- Сами изображения (например, `01.jpg`, `02.jpg`, и т.д.)

## Устранение проблем

Если анимация не работает:
1. Проверьте консоль браузера на наличие ошибок
2. Убедитесь, что файлы изображений существуют
3. Проверьте, что сервер запущен и файлы доступны
4. Проверьте путь к файлам в `files.json`
